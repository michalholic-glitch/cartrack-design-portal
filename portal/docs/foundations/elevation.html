Elevation

This page will cover surface depth: the shadow scale, when a surface raises (menus, dialogs, snackbars), and the z-index layering rules.

Not yet defined: tokens.json contains no shadow or z-index tokens — the product uses MUI's default shadow scale, unoverridden. That's the same situation fontFamily.muiDefault was in before it was documented, and extracting the shadow scale the same way is the logical next step — but it's a token-extraction task, not a docs task, so this page stays a stub until it happens.
