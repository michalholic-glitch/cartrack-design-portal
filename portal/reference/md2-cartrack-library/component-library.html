Cartrack — Material 2 Component Library

Scope: desktop web only. The Material 2 components needed for a desktop fleet portal, rendered with Cartrack brand tokens — primary orange #F47735, Roboto, 14px base, classic MD2 styling (4px corners, uppercase buttons, filled text fields). Mobile-only patterns are intentionally excluded; a few desktop-web staples MD2's mobile-first catalogue under-emphasizes (breadcrumbs, pagination, steppers, expansion panels) are added.

Excluded as mobile-only: bottom app bar, bottom navigation, bottom sheets, backdrop, and the floating action button (FAB). On desktop the primary action lives in the top app bar / page toolbar as a contained button, and primary navigation is the persistent drawer or rail — not bottom nav.

Accessibility flag: white text on the orange primary is ~2.79:1 (below WCAG AA); for filled buttons and the app bar carrying text, use primary-dark #BB4800 or dark text. Worth a deliberate decision.
